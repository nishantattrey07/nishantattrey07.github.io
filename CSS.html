What is CSS?


    CSS stands for Cascading Style Sheets, it is used to define and style HTML elements. In layman terms if HTML is a person,
    then CSS is the clothing of that person.

Syntax of CSS

Syntax: 

            h1 {
            color: green;
            text-align: right;
          }
        
    In the above example:-

    p = selector
    color = property 
    green = property value
    text-align = property
    right = property value

CSS Background

Plain Color

Syntax: 

            body {
                background-color: lightblue;
              }

Image

Syntax: 

            body {
                background-image: url("bi.png");
              }
        
    In the above example:-

    body = selector (As the whole page needs to be covered)
    background-image = Sets background image for an element
    url("bi.png") = The link of the image

CSS Border

CSS Border property is used to give a border to an element and also specifies different properties of the border.

Syntax:

            p.dotted {border-style: dotted;}
            p.dashed {border-style: dashed;}
            p.solid {border-style: solid;}
            p.double {border-style: double;}
            p.groove {border-style: groove;}
            p.ridge {border-style: ridge;}
            p.inset {border-style: inset;}
            p.outset {border-style: outset;}
            p.none {border-style: none;}
            p.hidden {border-style: hidden;}
            p.mix {border-style: dotted dashed solid double;}
        
    There are many different types of styles for borders stated in the above example
    border-style = Used to call different types of styles

CSS Margins

Margins is CSS is used to add space around the elements.

Syntax: 

            h1 {
                margin-top: 500px;
                margin-bottom: 100px;
                margin-right: 150px;
                margin-left: 80px;
              }
        
    In the above example:- 

    margin-top = Adjusts the top margin of an element with the specified value
    margin-bottom  = Adjusts the bottom margin of an element with the specified value
    margin-right = Adjusts the right margin of an element with the specified value
    margin-left = Adjusts the left margin of an element with the specified value

   
    #Important 

    auto can be used as a value as well, it automatically centers the element horizontally.

CSS Padding

In simple terms, Padding is used for spacing inside a defined div element

Syntax:

            div {
                padding-top: auto;
                padding-right: auto;
                padding-bottom: auto;
                padding-left: auto;
              }
        
    In the above example:- 

    padding-top = Adjusts the top padding of an element with the specified value
    padding-bottom  = Adjusts the bottom padding of an element with the specified value
    padding-right = Adjusts the right padding of an element with the specified value
    padding-left = Adjusts the left padding of an element with the specified value

CSS Height and Width

Height and Width properties of CSS are used to adjust the height and width of elements

Syntax

            div {
                height: auto;
                width: auto;
             }
        
    In the above example:-

    height = Adjusts the height of the element with the specified value
    widhth = Adjusts the width of the element with the specified value

CSS Box Model

The CSS box model comes into picture when the designing or layout of elements are concerned

Syntax:

            div {
                width: 300px;
                border: 15px solid green;
                padding: 50px;
                margin: 20px;
              }
        
    The above example is the way of designing elements

CSS Table

A table made in HTML can be made pleasing to the eye using table properties of CSS

Syntax:

            table, th, td {
                border: 1px solid;
              }
        
    In the above example:-

    Table along with table header and table data are taken and are given a border.

CSS Z-Index

In simple terms Z-Index in CSS is used to avoid overlapping of elements

Syntax:

            img {
                position: absolute;
                left: auto;
                top: auto;
                z-index: -1;
              }
        
    The above example is a basic way of defining an element with z-index.
    The z-index is intially at 0, it can be adjusted according to the which element the user wants on top.
